# Reward Distributor (Anchor)

A generic SPL-token reward distribution program.

## Features
- Authority-controlled reward allocation
- Claim-once enforcement
- PDA-secured vault
- Independent of staking logic

## Use Cases
- Staking rewards
- DePIN incentives
- Payroll
- Bounties
- DAO rewards

## Flow
1. Initialize pool
2. Fund pool
3. Allocate rewards
4. Users claim rewards
